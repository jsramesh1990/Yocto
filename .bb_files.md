# BitBake Recipe Files (.bb Files)

# Overview

A `.bb` file (BitBake recipe) is the fundamental building block of the:
- Yocto Project
- Poky
- OpenEmbedded

A `.bb` file tells BitBake:

```text
WHAT to build
WHERE to get source code
HOW to build it
HOW to install it
HOW to package it
```

Without `.bb` files:

```text
BitBake cannot build software.
```

---

# 1. What is a .bb File?

A `.bb` file is:

```text
A BitBake recipe file
```

used to describe how a software package should be built.

---

# 2. Why .bb Files are Needed

BitBake does not know:

- where source code exists
- dependencies
- compiler options
- installation paths

The recipe provides this information.

---

# 3. High-Level Workflow

```text
.bb File
    ↓
BitBake Parser
    ↓
Task Generation
    ↓
Cross Compilation
    ↓
Package Creation
    ↓
Root Filesystem
```

---

# 4. Recipe Naming Convention

Typical format:

```text
<package-name>_<version>.bb
```

Examples:

```text
busybox_1.36.1.bb
bash_5.2.bb
openssl_3.0.bb
```

---

# 5. Recipe Location

Recipes are stored inside layers.

Example:

```text
meta-custom/
└── recipes-apps/
    └── hello/
        └── hello_1.0.bb
```

# 31. Recipe Tasks

Every recipe generates tasks.

---

# 32. Main Tasks

| Task | Purpose |
|--------|----------|
| do_fetch | Download |
| do_unpack | Extract |
| do_patch | Apply patches |
| do_configure | Configure |
| do_compile | Compile |
| do_install | Install |
| do_package | Package |

---

# 33. Complete Task Flow

```text
do_fetch
     ↓
do_unpack
     ↓
do_patch
     ↓
do_configure
     ↓
do_compile
     ↓
do_install
     ↓
do_package
```

---

# 34. Custom do_compile()

Example:

```bitbake
do_compile()
{
    gcc hello.c -o hello
}
```

---

# 35. Cross Compilation Version

Preferred:

```bitbake
do_compile()
{
    ${CC} hello.c -o hello
}
```

---

# 36. Why Use ${CC}?

Uses Yocto cross compiler automatically.

# 65. Recipes vs Makefiles

| Recipe | Makefile |
|----------|-----------|
| High-level metadata | Build commands |
| Dependency aware | Local project |
| Package generation | No package support |
| Cross compile integrated | Manual setup |

---

# 68. Summary

- `.bb` files are BitBake recipes
- Define source, dependencies, build and install steps
- Stored inside Yocto layers
- Generate tasks such as do_fetch, do_compile, do_install
- Support cross-compilation automatically
- Can inherit classes like autotools and cmake
- Fundamental building blocks of Yocto and Poky
